# Plan de competences

## Produit et metier

- Comprendre les statuts d'exercice paramedical
- Comprendre BNC, micro-BNC, declaration controlee et notions de charges deductibles
- Connaitre les contraintes RGPD et donnees de sante
- Savoir dialoguer avec un expert-comptable

## Frontend

- TypeScript strict
- React et Next.js App Router
- Formulaires robustes avec validation
- Accessibilite clavier et lecteur d'ecran
- Etats de chargement, erreur et vide
- Tableaux filtrables et exportables

## Backend

- Modelisation relationnelle PostgreSQL
- Prisma migrations
- Validation Zod
- RBAC et multi-tenancy
- Journalisation d'audit
- Files/jobs pour exports lourds

## Qualite

- Tests unitaires Vitest
- Tests integration API
- Tests e2e Playwright
- Donnees de test anonymes
- Revue de securite sur modules sensibles

## Securite et conformite

- Authentification forte
- Sessions securisees
- Chiffrement au repos selon hebergeur
- Sauvegardes et restauration
- Minimisation des donnees
- Droit d'acces, rectification, suppression
- Registre des traitements

## DevOps

- Environnements dev/staging/prod
- Variables d'environnement
- CI avec lint, test et build
- Migrations controlees
- Monitoring erreurs
- Backups automatises
